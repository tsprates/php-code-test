# LifeRaft Code Test

## Installation

* Clone the project:

```sh
git clone https://github.com/tsprates/php-code-test
```

* Build the project:

```sh
docker-compose build
```

* Start the project:

```sh
docker-compose up -d
```

> Note: The file `customers.txt` can be found in the directory `API/src/storage/public/`.
